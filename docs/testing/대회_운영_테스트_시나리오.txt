3. 조별리그 경기 수 및 소요 시간
----------------------------------------------------------------

[3-1. 남자부]

  구분         경기 수               소요 시간
  ----------   -------------------   ----------
  5명 조 (×8) 조당 10경기 × 8조    = 80경기
  4명 조 (×2) 조당 6경기 × 2조     = 12경기
  합계                               92경기 × 25분

  ※ 경기장 4개 병행 운영 기준

  4명 조 (×2) 조당 6경기 × 2조     = 12경기
  합계                               22경기 × 25분

6. 테스트 시나리오 체크리스트
----------------------------------------------------------------

[시나리오 1] 조 편성 및 대진표 생성
  □ 남자 42명 → 10개조 자동 배분 (5명×8, 4명×2) 정상 처리
  □ 여자 13명 → 3개조 배분 정상 처리
  □ 각 조별 대진표(라운드로빈) 자동 생성 확인
  □ 경기장 배정 (남자: 1~4번 / 여자: 5~6번) 정상 반영

[시나리오 2] 조별리그 운영
  □ 경기 시작 09:30 설정 및 시간표 자동 생성 확인
  □ 조별리그 경기당 25분 단위 스케줄 정상 생성
  □ 4개 경기장 병행 시 충돌(선수 중복 출전) 없는지 확인
  □ 동점 시 순위 결정 기준 적용 확인 (득실, 골득실 등)

[시나리오 3] 본선 진출자 선별
  □ 남자부: 각 조 1위 10명 자동 본선 진출 처리
  □ 남자부: 각 조 2위 중 상위 6명 선별 로직 확인
  □ 여자부: 각 조 1·2위 6명 자동 본선 진출 처리
  □ 여자부: 각 조 3위 중 상위 2명 선별 로직 확인
  □ 동점·동성적 발생 시 타이브레이크 처리 확인

[시나리오 4] 토너먼트 운영
  □ 16강 대진 자동 생성 및 시드 배정 확인 (남자부)
  □ 8강 대진 자동 생성 확인 (여자부)
  □ 토너먼트 경기당 45분 단위 스케줄 정상 생성
  □ 여자부 결승 16:00~16:45 / 5번 경기장 고정 배정 확인
  □ 18:00 이전 전 경기 완료 가능 여부 일정 시뮬레이션

[시나리오 5] 심판 배치
  □ 심판 12명 풀에서 경기별 1명 또는 2명 임의 배치
  □ 동시 진행 경기 수 초과 배치 시 경고 처리 확인
  □ 결승전 심판 2명 배치 권장 알림 확인
  □ 심판 중복 배치 (동시간대 2경기) 방지 로직 확인

[시나리오 6] 예외 상황 처리
  □ 선수 기권·부상 발생 시 대진표 재편성 처리
  □ 경기 지연 발생 시 이후 경기 시간 자동 조정 확인
  □ 경기장 사용 불가 시 대체 경기장 배정 처리
  □ 18:00 종료 시간 초과 예상 시 알림 처리
